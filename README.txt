WHM Timer — Final PWA

Что загружать на GitHub Pages:
- index.html
- manifest.webmanifest
- sw.js
- папку assets целиком

Структура должна сохраниться:
WHM/
├─ index.html
├─ manifest.webmanifest
├─ sw.js
└─ assets/
   ├─ icon-192.png
   ├─ icon-512.png
   ├─ apple-touch-icon.png
   ├─ ambient.mp3 / ambient.ogg
   ├─ final-cycle.mp3 / final-cycle.ogg
   ├─ soft-gong.mp3 / soft-gong.ogg
   └─ completion-gong.mp3 / completion-gong.ogg

Логика:
1) 30 циклов: фоновый звук на 1–29, отдельный звук на 30-м.
2) Задержка после выдоха: нажать экран, чтобы зафиксировать время.
3) Глубокий вдох: 3 секунды, без сигнала.
4) Удержание: 20 секунд, фраза «Задержи дыхание», без сигнала на старте.
5) В конце 20 секунд: длинный гонг.
6) Пауза: 5 секунд. Затем новый раунд автоматически начинается с экрана 1/30.
7) Практику можно завершить в паузе после любого законченного раунда.

Заметка:
GitHub Pages после обновления может показать прежнюю версию из кэша PWA.
Для проверки обновления на iPhone закрой приложение, затем открой ссылку в Safari и обнови страницу.
